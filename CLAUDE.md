## Project Overview

The Finance Intelligence Platform automates the ingestion, validation,
analysis and reporting of financial data using AI-assisted workflows.

Primary goal:

Reduce manual spreadsheet work while producing reliable executive insights.

---

## Engineering Principles

Prioritize:

- Reliability
- Explainability
- Maintainability
- Business value
- Clear documentation

---

## Coding Style

When suggesting code:

- Prefer readability over cleverness.
- Explain architectural trade-offs.
- Preserve existing workflow behaviour unless requested.
- Add comments only where they improve understanding.
